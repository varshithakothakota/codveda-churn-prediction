# Customer Churn Prediction

This project was completed as part of the Codveda Technology Business Analytics Internship.

## Project Overview
Customer churn prediction helps companies identify customers who are likely to leave a service.

In this project, we analyze telecom customer data and build a machine learning model to predict churn.

## Tools Used
Python
Pandas
Seaborn
Scikit-Learn
Google Colab

## Model Used
Random Forest Classifier

## Results
The model successfully predicts customer churn with high accuracy.

## Business Impact
Businesses can use churn prediction models to identify at-risk customers and take actions such as offering discounts or improving customer support.
